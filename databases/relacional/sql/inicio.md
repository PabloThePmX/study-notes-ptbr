# Inicio

* O Banco de Dados Relacional é mais usado atualmente, ele armazena dados estruturados e é organizado em tabelas.
* Representa da mesma maneira que o Excel (colunas e linhas, uma matriz)
* As tabelas podem se relacionar entre si.
* MySQL é uma opção grátis, porém não tão forte como SQL Server e Oracle Database.
* Estruturado significa que cada dado tem um tipo, e por isso vai aceitar valores daquele tipo.
* Basicamente a tabela segue uma regra rígida, se o valor não for do tipo especificado, vai dar erro.
* As tabelas se relacionam entre si usando o ID do registro que será buscado da outra tabela.
* No SQL temos 5 categorias de comandos: DDL, DCL, DML, TCL e DQL.
* DBMS (Database Management System) é um software utilizado para acessar, manipular e monitorar um sistema de banco de dados.
  * É como se fosse uma IDE.
* No SQL é sempre aspas simples
* Lembrar que a FK é basicamente um apontamento (referência) para outra tabela, nesta outra há a informação que queremos.
  * A FK sempre vai ficar na tabela "many".
