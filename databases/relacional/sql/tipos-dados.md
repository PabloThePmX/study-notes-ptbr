# Tipos de Dados

## String
* `char` e `varchar` são os principais
  * `char`: bom para dados que tem um tamanho certo, como siglas de estados que SEMPRE tem 2 caracteres (número fixo de tamanho de um char)
  * `varchar`: quando não sabemos o tamanho da cadeia de caracteres, então definimos só qual poderá ser seu tamanho máximo (quando não tem um número fixo de tamanho do char)

## Numérico
  * O `bit` é um booleano (0, 1 ou NULL)
  * `int`, `bigint` ou `decimal` (para valores monetários)

## DateTime
  * `Datetime1` e `Datetime2` muda o ano de início
  * O 2 é melhor para usar
